# Action 目录说明

这个目录按 action 拆分 `workspace-sync` 的主要操作说明，便于按命令单独查阅。

当前包含：

- `push.md`：保存当前工作空间到远端
- `pull.md`：从远端恢复一个工作空间
- `list.md`：查看当前后端中已有的工作空间
- `clean.md`：删除指定工作空间或清理全部工作空间
- `install.md`：安装 skill 到本地 Claude Skills 目录
- `uninstall.md`：从本地移除已安装的 skill

建议阅读顺序：

1. 先看 [`../docs/usage-guide.md`](../docs/usage-guide.md) 了解整体流程
2. 再按具体命令查看对应 action 文档
